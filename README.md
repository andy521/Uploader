#Uploader6.0 上传控件代码重构<br/>
1.不再支持IE9及以下浏览器<br/>
2.前后文件读取使用FileReader<br/>
3.前台图片处理使用Canvas<br/>
4.前后台数据交货使用WebSocket，后台逻辑分装在Asp.Net平台下<br/>
#功能说明：<br/>
1.支持大文件上传<br/>
2.支持多个文件上传<br/>
3.支持前台图片剪切<br/>
4.支持前台图片压缩（大小图模式）<br/>
